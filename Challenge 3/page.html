HTML (HyperText Markup Language)

Créer votre site Web avec HTML5 et CSS3

Un lien hypertexte est un élément placé dans le contenu d'une page Web et qui permet, en cliquant dessus, d'accéder à un autre contenu sur le même site Web (lien interne) ou à un site Web différent (lien externe). On le nomme également hyperlien Le lien hypertexte prend la plupart du temps la forme d'un texte. On le repère alors facilement puisque l'usage veut que ce texte apparaisse (CSS) et d'une autre couleur, le bleu dans 99 % des cas. Mais il peut également être placé sur une image,(Java script) .

Le lien hypertexte fait partie des éléments de base du langage HTML et, plus largement, de la navigation sur le Web. Le terme de Toile , employé pour parler du Web, fait d'ailleurs référence à cette multiplication de liens qui s'orientent dans de nombreuses directions, pour offrir aux internautes la possibilité d'explorer le World Wide Web dans son intégralité. On peut utiliser le lien hypertexte pour des raisons marketing, pour le référencement d'un site Web, etc.
